# WBS Tracker Project

## 📝 Introduction
This project is a Work Breakdown Structure (WBS) Tracker dashboard built with Python and Streamlit. It allows teams to monitor, manage, and track project progress dynamically by syncing data directly with a Google Sheets database (`Data_UAT`).

## 🚀 How to Run and Use Local

### 1. Prerequisites
Ensure you have Python 3.9+ installed on your system.

### 2. Installation
git clone https://github.com/ginnybui/wbs-tracker.git
cd wbs-tracker
pip install -r requirements.txt
